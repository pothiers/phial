toolbox.all_states(N, spn=2, backwards=False)¶

All combinations spn^N binary states in lexigraphical order. This is NOT the order used in most IIT papers. To get order for papers, set ‘backwards=True’. RETURN list of statestr in hex format. e.g. ‘01100’ spn:: States Per Node
